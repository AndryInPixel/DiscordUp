Discord Up commands list!


Method: <Required> , [optional]



▶ Moderation Commands


ds/help : Gets information about the commands

ds/kick <@user> [reason] : kicks an user

ds/ban <@user> [reason] : Bans an user

ds/unban <Username#Discriminator> : Unbans an user in the Ban Entry

ds/warn <@user> [reason] : Warns an user

ds/clear [amount] : Deletes a given number of messages (Default is 1)

ds/welcome : Sets up a welcome screen

ds/welcome <#channel> : Sets the welcome channel (This is where the bot is going to send the message)

ds/welcome <Text> : Sets the welcome text (This is what your users will receive when they join the server)

ds/welcome disable : Disables the welcome screen

▶ Utility Commands


ds/userinfo <@user> : Gets the info of a user

ds/serverinfo : Gets the info of the server

ds/poll <"description"> <Option1> or <Option2> : Creates a poll (Be careful with the syntax, otherwise it won't work properly)

ds/reddit [subreddit] : Gets a random post from reddit (Default subreddit is discordapp)

ds/roll [number] : Rolls a random number within a given limit (Default is 20)

ds/invite : Sends the bot invite

ds/avatar <@user> : Gets the avatar of an user

ds/membercount : Gets the whole number of members in a server (Does not include bots)

ds/base64 <encode/decode> <Normal Text/Base64 Text> : Encodes or decodes in base64

ds/addreaction <#channel> <message id> <emoji> <@role> : Adds a reaction to a message that gives a role on click

ds/removereaction <message id> <emoji> : Removes the reaction role


▶ Misc Commands


ds/about : Tells you the basic information about the Bot

ds/cat : Sends random pictures of cats

ds/dog : Sends random pictures of dogs

ds/fox : Sends random pictures of foxes

ds/racoon : Sends random pictures of racoons

ds/panda : Sends random pictures of Pandas

ds/ship <option1> <option2> : Ships 2 objects with a random percentage

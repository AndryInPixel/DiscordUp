Discord Up commands list!


Method: <Required> , [optional]



▶ Moderation Commands


?help : Gets information about the commands

?kick <@user> [reason] : kicks an user

?ban <@user> [reason] : Bans an user

?unban <Username#Discriminator> : Unbans an user in the Ban Entry

?warn <@user> [reason] : Warns an user

?clear [amount] : Deletes a given number of messages (Default is 1)

?welcome : Sets up a welcome screen

?welcome <#channel> : Sets the welcome channel (This is where the bot is going to send the message)

?welcome <Text> : Sets the welcome text (This is what your users will receive when they join the server)

?welcome disable : Disables the welcome screen

▶ Utility Commands


?userinfo <@user> : Gets the info of a user

?serverinfo : Gets the info of the server

?poll <"description"> <Option1> or <Option2> : Creates a poll (Be careful with the syntax, otherwise it won't work properly)

?reddit [subreddit] : Gets a random post from reddit (Default subreddit is discordapp)

?roll [number] : Rolls a random number within a given limit (Default is 20)

?invite : Sends the bot invite

?avatar <@user> : Gets the avatar of an user

?membercount : Gets the whole number of members in a server (Does not include bots)

?base64 <encode/decode> <Normal Text/Base64 Text> : Encodes or decodes in base64

?addreaction <#channel> <message id> <emoji> <@role> : Adds a reaction to a message that gives a role on click

?removereaction <message id> <emoji> : Removes the reaction role
